题目：点 A、B 在数轴上，点 A 是 ﹣17，点 B 是 15，线段 AB 的长是 32；PQ 是一条定长的线段（点 P 在点 Q 的左侧），它在数轴上从左向右匀速运动，线段 PQ 的长为 8；线段 PQ 运动的速度为 2 个单位长度/秒；当动线段 PQ 运动到点 Q 与点 A 重合时，与此同时，点 C 从点 P 出发，在动线段 PQ 上，以 1 个单位长度/秒的速度向点 Q 运动，遇到点 Q 后，点 C 立即原速返回，向点 P 运动，遇到点 P 后也立即原速返回，向点 Q 运动。设动线段 PQ 以及点 C 同时运动的时间为 t 秒（0 < t ≤ 20），当 4PC − QB = 4 时，求 t 的值。
P 坐标-25.00P = -25+2t
Q 坐标-17.00Q = -17+2t
C 坐标-25.00C = P+PC
📏 PC0.00PC = t
📏 QB32.00QB = 32−2t
⭐ 4PC − QB-32.00
P（红色） Q（蓝色） C（绿色，折返） ▎ A（﹣17） ▎ B（15）
